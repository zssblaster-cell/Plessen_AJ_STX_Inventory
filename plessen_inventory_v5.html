Setup Guide — Shared Office Access

Access from any device in the office with shared, real-time data.

Note: Data currently lives in this browser only. The options below fix that.

Easiest Option 1 — Dedicated Office Terminal

Save this HTML file to a shared drive. Use one office computer as the inventory terminal — all staff update on that machine. Data persists in its local storage.

Recommended Option 2 — Vercel (Free, 10 min)

Publish a free URL like plessen-inventory.vercel.app — works on any phone or computer.

1

Create free account at vercel.com

Sign up with Google — takes 2 minutes.

2

Upload this HTML file

Dashboard → Add New Project → drag and drop. Publishes instantly.

3

Bookmark the URL for all staff

Moderate Option 3 — Firebase Real-Time Sync

All devices see the same inventory live. Best for multiple staff updating simultaneously.

1

Create project at console.firebase.google.com

Name it plessen-inventory. Enable Firestore in test mode.

2

Get your Firebase config, share in this chat

const firebaseConfig = { apiKey: "AIza…", projectId: "plessen-inventory" };

Say "Help me connect Firebase" and we'll generate the synced version.

Requires IT Option 4 — On-Premise Server

1

Install Node.js (nodejs.org)

2

Run in the HTML file folder:

npx serve .

Access at http://YOUR-PC-IP:3000 on all office devices.

Need help? Return to this chat and say "Help me set up Firebase" or "Help me host on Vercel" — we'll walk through it and generate the connected version.
